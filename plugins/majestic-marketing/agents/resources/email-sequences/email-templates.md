# Email Structure Templates

Copy frameworks for each email in a sequence. Use these as scaffolding when writing full email copy.

## Welcome Sequence Templates

### Email 1: Delivery (Immediately)

**Purpose:** Deliver lead magnet, set expectations, get micro-engagement.

```
[Greeting — keep it simple]
[Deliver the goods — link]
[Quick start — one action in 5 minutes]
[Set expectations — what emails coming]
[Micro-CTA — hit reply, answer a question]
[Sign off]
```

### Email 2: Connection (Day 2)

**Purpose:** Build rapport through vulnerability.

```
[Story hook — specific moment]
[The struggle — what you went through]
[The insight — what you learned]
[Connection — how this relates to them]
[Soft forward reference — hint at tomorrow]
```

### Email 3-4: Value (Days 4, 6)

**Purpose:** Teach something useful, demonstrate expertise.

```
[Hook — insight or observation]
[The problem — what most people get wrong]
[The solution — what to do instead]
[Example or proof — show it working]
[Action step — what they can do now]
```

### Email 5: Bridge (Day 8)

**Purpose:** Show gap between current state and possibility.

```
[Acknowledge progress — what they can now do]
[Reveal the gap — what they still can't do]
[Paint the picture — what's possible with full solution]
[Soft mention — the offer exists, no hard sell]
```

### Email 6: Soft Pitch (Day 10)

**Purpose:** Introduce offer properly, let them self-select.

```
[The offer — what it is, what's included]
[Who it's for — specific situations]
[Who it's NOT for — disqualification]
[Social proof — if available]
[Soft CTA — no urgency yet]
```

### Email 7: Direct Pitch (Day 12)

**Purpose:** Make the clear ask.

```
[Direct opener — no buildup]
[Restate core value — one sentence]
[Handle remaining objection]
[Urgency — if real]
[Clear CTA]
```

## Conversion Sequence Templates

### Email 1: Introduce Offer

```
[Hook — problem or desire]
[Core promise — what the offer delivers]
[What's included — bullet points]
[Who it's for]
[CTA — learn more or buy]
```

### Email 2: Paint Transformation

```
[Before state — their current reality]
[After state — what's possible]
[The gap — why they can't bridge it alone]
[The offer — how it bridges the gap]
[CTA]
```

### Email 3: Social Proof

```
[Lead with result — specific outcome]
[The story — who achieved it, their situation]
[The process — what they did]
[The result — measurable improvement]
[Connect to reader — "you can too"]
[CTA]
```

### Email 4: Objection Handler

```
[Name the objection — don't dance around it]
[Validate it — show you understand]
[Reframe or resolve]
[Evidence — proof it's not an issue]
[CTA]
```

### Email 5-6: Urgency + Final Push

```
[Deadline reminder — specific time]
[What they're missing — pain of inaction]
[Quick recap — core value]
[Clear CTA — one action]
```

### Email 7: Last Call

```
[Final notice — hours remaining]
[One-line value prop]
[Single CTA]
[P.S. — deadline is real]
```

## Launch Sequence Templates

### Pre-Launch: Seed Interest

```
[Tease what's coming — without full reveal]
[Why it matters — problem it solves]
[When to expect it]
[Optional: waitlist or early access CTA]
```

### Cart Open: Announcement

```
[It's live — clear statement]
[What it is — full description]
[What's included — complete list]
[Pricing — transparent]
[Bonuses — if applicable]
[CTA — buy now]
```

### Mid-Launch: Case Study

```
[Specific customer — name, situation]
[Their challenge — relatable problem]
[What they did — used your offer]
[Results — specific, measurable]
[Connect to reader]
[CTA]
```

### Cart Close: Final Hours

```
[Time remaining — specific]
[What goes away — scarcity is real]
[Last chance reminder]
[Single CTA]
```

## Re-engagement Templates

### Email 1: Pattern Interrupt

```
[Different tone — break the pattern]
[Acknowledge absence — no judgment]
[Remind value — what they signed up for]
[Simple ask — are you still interested?]
```

### Email 2: Pure Value

```
[Best content — no selling]
[Genuinely helpful — solve a problem]
[No ask — just value]
[Optional: "more like this coming"]
```

### Email 3: Direct Question

```
[Honest ask — do you want to stay?]
[Easy out — unsubscribe is okay]
[Value reminder — what they'll miss]
[Binary CTA — stay or go]
```

### Email 4: Final Notice

```
[Removing from list — specific timeframe]
[Last chance to stay]
[Simple click to remain]
[Respect their choice]
```
