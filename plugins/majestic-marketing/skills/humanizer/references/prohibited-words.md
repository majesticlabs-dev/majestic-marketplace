# Prohibited Words & Phrases

Words and phrases that AI detectors flag and human readers recognize as machine-generated. Replace or remove every instance.

## Banned Words

|--------|-------------|

| navigating | working through, handling, dealing with |
| complexities | problems, challenges, details |
| realm | area, field, space |
| bespoke | custom, custom-built |
| tailored | custom, fitted, adapted |
| towards | toward (or rewrite) |

| ever-changing | changing, shifting |
| ever-evolving | changing, growing |
| daunting | hard, intimidating, difficult |
| amongst | among |
| robust | strong, solid, reliable |

## Banned Phrases

### Never Open With
- "In today's [landscape/world/era]..."
- "In the realm of..."
- "In the heart of..."
- "The world of..."
- "When it comes to..."
- "It's important to note that..."
- "It is advisable..."
- "Let's delve into..."
- "Let's explore..."

### Never Close With
- "In conclusion..."
- "In summary..."
- "To summarize..."
- "Overall, [subject] continues to be..."

### Never Use (Compound Phrases)
- "not only... but also..."
- "seeking more than just..."
- "designed to enhance..."
- "it's not merely..."
- "our suite..."
- "unlock the secrets..."
- "unveil the secrets..."

### Engagement Bait (Remove Entirely)
- "Let that sink in"
- "Read that again"
- "Full stop."
- "This changes everything"
- "Are you paying attention?"

### Sycophantic (Remove Entirely)
- "I hope this helps!"
- "Great question!"
- "That's an interesting point."

### Vague Attribution (Replace With Specifics)
- "Some experts believe..."
- "Many scholars argue..."
- "Research shows..."
- "It is widely regarded as..."
- "It is generally accepted..."

---

## The Cluster Rule

A single banned word is weak evidence. **Three or more in one paragraph is a rewrite trigger.** Scan each paragraph as a unit, not individual sentences.
