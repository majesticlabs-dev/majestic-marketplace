# Majestic Sales

Sales acceleration and revenue tools. Includes 1 command and 6 skills.

## Installation

```bash
/plugin install majestic-sales
```

## Quick Reference

| Task | Invocation |
|------|------------|
| Build sales funnel | `/majestic-sales:workflows:funnel-builder` |
| Create sales playbook | `skill majestic-sales:sales-playbook` |
| Design outbound sequences | `skill majestic-sales:outbound-sequences` |
| Write proposals | `skill majestic-sales:proposal-writer` |
| Define ideal customer | `skill majestic-sales:icp-discovery` |
| Design referral program | `skill majestic-sales:referral-program` |
| Create win-back campaign | `skill majestic-sales:win-back` |

## Commands

### workflows

Invoke with: `/majestic-sales:workflows:<command>`

| Command | Description |
|---------|-------------|
| `funnel-builder` | Build comprehensive, high-converting sales funnels through a guided 5-phase process |

## Skills

### Sales Enablement

| Skill | Description |
|-------|-------------|
| `sales-playbook` | Create comprehensive sales playbooks with discovery frameworks, objection handling, competitive positioning, demo scripts, and closing techniques |
| `proposal-writer` | Create winning sales proposals and SOWs with executive summaries, scope of work, pricing strategies, and closing elements |

### Prospecting

| Skill | Description |
|-------|-------------|
| `outbound-sequences` | Design cold outreach sequences for email and LinkedIn with personalization frameworks and follow-up cadences |
| `icp-discovery` | Systematically discover and define your Ideal Customer Profile with firmographic criteria and buyer personas |

### Growth & Retention

| Skill | Description |
|-------|-------------|
| `referral-program` | Design viral referral programs with incentive structures and sharing mechanics to turn customers into advocates |
| `win-back` | Design win-back campaigns to re-engage dormant customers and recover churned users |
