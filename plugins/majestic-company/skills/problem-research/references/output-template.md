## Section Guidelines

### Executive Summary
- Maximum 4 sentences
- Lead with the most important finding
- Include the verdict
- Make it actionable

### Pain Points Table
- Include actual quotes (verbatim where possible)
- Note the source platform
- Calculate PPS accurately
- Sort by PPS descending

### Must-Haves
- Focus on switching blockers
- Count competitor coverage accurately
- Explain why each is non-negotiable

### Hidden Gems
- Require 3+ evidence points each
- Be honest about opportunity size
- Note actionability level

### Viability Assessment
- Score each factor with specific evidence
- Don't inflate scores
- Be brutally honest in the narrative
- Make kill criteria measurable

### Quote Selection
- Prefer specific over vague
- Include user context when available (role, company size)
- Favor quotes that reveal underlying needs
- Avoid promotional or obviously fake reviews
