The one thing the user was taking for granted that made this failure possible. State it in a single sentence.

1-2 concrete, observable signals the user could watch for that would indicate this failure mode is starting to play out. These must be things you can actually see or measure (a metric, a behavior, a missed milestone) — not vague feelings.

RULES
- Be direct. Don't hedge. Don't sugarcoat.
- Don't pad. If 200 words tells the story, stop at 200.
