# Article Writing SOP

Standard operating procedure for writing articles using the Majestic Marketing toolkit.

## Phase 1: Research & Planning

- `/keyword-research` — Find and prioritize topics using the 6 Circles Method
  - Use first when you don't have a topic yet
- `/bofu-keywords` — Find high-intent keywords if targeting buyers ready to convert
- `/seo:content-planner` — Create content outline and topic cluster map
  - Structures headers, subtopics, and internal linking plan

## Phase 2: Voice Setup (one-time)

- `/style-forensics` — Analyze a writing sample to extract your Style DNA
  - Feed it 2-3 existing articles you've written
- `/brand-voice` — Codify organizational voice into reusable guide
  - Or use `/style-guide-builder` for broader editorial standards

## Phase 3: Writing

- `/content-writer` — Write the article from scratch (outline-first workflow)
  - Best for general articles, guides, blog posts
- `/seo-content` — Write SEO-optimized content that ranks AND reads human
  - Use instead of `content-writer` when SEO is the primary goal
- `/style-writer` — Write matching your Style DNA report
  - Requires a `style-dna.md` from Phase 2

## Phase 4: Optimization

- `/content-optimizer` — Improve keyword integration, structure, semantic depth
- `/geo-content-optimizer` — Optimize for AI citation (ChatGPT, Perplexity, Gemini)
- `/meta-optimizer` — Generate meta title, description, URL slug
- `/snippet-hunter` — Format for featured snippets / SERP features
- `/schema-architect` — Generate JSON-LD structured data
- `/structure-architect` — Optimize header hierarchy and internal linking

## Phase 5: Quality Review

- `/copy-editor` — Grammar, style, clarity pass
- `/humanizer` — Strip AI patterns if content sounds robotic
- `/fact-checker` — Verify statistics, quotes, claims
- `/content-auditor` — Score quality, E-E-A-T signals, SEO compliance

## Phase 6: Repurpose (optional)

- `/content-atomizer` — Turn the article into 15+ platform-ready assets
- `/thread-builder` — Convert into a Twitter/X thread

## Recommended Minimum Flow

1. `/keyword-research` — pick topic
2. `/seo-content` or `/content-writer` — draft
3. `/copy-editor` — polish
4. `/content-optimizer` — SEO pass
5. `/meta-optimizer` — metadata

## Notes

- Phase 2 is one-time setup — once you have a Style DNA or brand voice, reuse it across articles
- Phase 4 skills are selective — you don't need all of them every time
- `/seo-content` already bakes in optimization, so you may skip parts of Phase 4 when using it
- Phase 6 is for amplification after publishing
